# XPOZ Finance Skills for Claude.ai

15 skills with interactive React artifact visualizations.

## How to Use

1. Copy the contents of any `SKILL.md` file
2. Add to Claude.ai as a custom instruction or Project knowledge
3. Ask Claude questions that trigger the skill
