# Skills Quick Reference

## How to Use
1. Copy the SKILL.md content for your chosen skill
2. Open a new Claude.ai conversation
3. Paste the skill content
4. Ask using the example prompt below

---

## 01 - Velocity Tracker
**What it does:** Track real-time social buzz velocity for any ticker vs hourly baseline
**Example prompts:**
- "What's the velocity for NVDA right now?"
- "Is Tesla getting more or less attention than usual?"
- "Track mentions for Bitcoin"
- "Compare AAPL velocity to tech peers"

## Tips for Best Results

1. **Be specific with tickers**: "NVDA" works better than "Nvidia stock"
2. **Time context**: Add "today", "this week", "right now" for recency
3. **Comparisons**: Add "vs peers" or "compared to sector" for context
4. **Multi-asset**: List multiple tickers for portfolio-style analysis
